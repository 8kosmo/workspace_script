시맨틱 웹

            시맨틱 태그는 모두 div태그와 같은 기능을 수행하는 태그임.
            검색엔진이나 그 이외의 기계적인 동작들이 웹페이지에 쉽게
            이해 할 수 있도록 제공됨.
            html5이전에는 화면레이아웃에 table태그를 주로 사용함.
            html5로 오전서 div태그를 주로 사용 -> ajax기술지원

                    footer - 페이지 하단 부분
